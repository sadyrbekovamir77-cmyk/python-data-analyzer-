# 📊 Python Data Analyzer Tool

## 🧠 About the project
This is a beginner Python project that analyzes a list of numbers and generates useful statistics.

## ⚙️ Features
- Accepts user input numbers
- Stops when user types "stop"
- Calculates:
  - Total sum
  - Average
  - Maximum value
  - Minimum value
  - Even / Odd analysis

## 💻 How to run
1. Run the Python file
2. Enter numbers one by one
3. Type "stop" to finish input
4. View the final report

## 🚀 Skills used
- Python basics
- Loops (while, for)
- Lists
- Functions (def)
- If / else logic

## 📌 Purpose
This project was built to practice Python and data analysis basics for beginners.
